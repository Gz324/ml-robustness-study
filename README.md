# Robustness of ML Models under Noisy Data

## Overview
This project analyzes how machine learning models behave under noisy input data.

## Models Used
- Logistic Regression
- Random Forest

## Methodology
- Added Gaussian noise to dataset at different levels
- Evaluated model performance using accuracy
- Compared robustness across models

## Results
- Observed performance degradation with increasing noise
- Random Forest showed better robustness than Logistic Regression

## Output
- results.png (visualization)
- results.csv (metrics)
